\# Futsal Formation Maker



フットサルのフォーメーションを作成するためのツールです。  

選手の配置を視覚的に管理し、戦術検討やチーム共有に活用することを目的としています。



---



\## アプリ概要



フットサルではフォーメーションが重要ですが、  

ホワイトボードや紙で管理するのは不便なことがあります。



このアプリでは、ブラウザ上で簡単にフォーメーションを作成できます。



---



\## 機能



・選手配置  

・ポジションの可視化  

・フォーメーション作成
